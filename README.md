🏠 Bengaluru House Price Prediction

A machine learning project to predict house prices in Bengaluru using data analysis and a trained model. This project includes data preprocessing, exploratory data analysis (EDA), model building, and a web application for predictions.

📌 Project Overview

This project aims to build an accurate house price prediction system using real estate data from Bengaluru. It leverages machine learning techniques to help users estimate property prices based on various features.

📂 Project Structure
├── Bengaluru_House_Data.csv.xls   # Raw dataset
├── cleaned_df.csv                # Cleaned dataset
├── EDA.ipynb                     # Exploratory Data Analysis
├── Rf_model.joblib               # Trained Random Forest model
├── app.py                        # Streamlit web application
├── house_logo.jpg.jpeg           # UI asset
├── Requirements.txt              # Dependencies
